# Аудио витрины пар (`pairCardAudio.ts`)

## Как услышать звук

1. **Один раз кликните или нажмите клавишу** по странице (политика автовоспроизведения).
2. Раскройте **«Пара N»** и наведите курсор на **область видео** (рамка с эффектами).

## Файлы в репозитории

Синтезированные **моно WAV** (44.1 kHz, 16-bit), ~2 с loop для ambient:

- `pair-01-ambient.wav` … `pair-10-hover.wav`

Перегенерация (без зависимостей, только Node):

```bash
cd mafia_sii5_ru && npm run generate:pair-audio
```

Скрипт: `scripts/generate-pair-audio.mjs` (чистый синтез: гармоники, FM, шум и т.д.). Если удалить WAV, снова включится **Web Audio fallback** из `pairCardAudio.ts`.
